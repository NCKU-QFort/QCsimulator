# Quantum Circuit (QC) Simulator 量子電路模擬器

[![Deploy](https://github.com/NCKU-QFort/QCsimulator/actions/workflows/deploy.yml/badge.svg?branch=main)](https://ncku-qfort.github.io/QCsimulator/)

Interactive quantum circuit simulator for education.  
互動式量子電路模擬器，適用於量子資訊課程教學。

## Local Build

```bash
npm install
npm run dev
```
